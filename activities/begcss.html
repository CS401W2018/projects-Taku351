Explain the difference between inline, embedded, and external CSS.

- Inline styles are added directly to HTML tags. Embedded styles are in the page's header and apply to the whole page. An external style sheet is a separate file that all pages on the site can link to for styling.

Explain why you should only use external CSS.

- External CSS makes pages load faster since the CSS file is downloaded once for all pages. It keeps HTML files clean and organized.

What is a selector used for?

A selector is used to target and style HTML elements or identify components in a component tree.
