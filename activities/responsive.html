Engagement questions

Why is the viewport meta tag important?

The viewport meta tag helps a webpage look right on different devices, especially mobile. Without it, text and images might be too big or too small.

How can you adjust the CSS for printing?

By using the Print CSS and place another link of CSS in the head.

When would you want to adjust the CSS for printing?

When a webpage needs to be properly formatted and optimized for print.
