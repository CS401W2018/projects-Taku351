Engagement questions

Why do we need a viewport meta tag?

The viewport meta tag helps web pages fit properly on all screens, especially mobile. Without it, websites may look too big or too small.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

If you set flex-direction: column; and flex-direction: row; for the same element, the one written last in the CSS file takes priority unless the first one is more specific or uses !important.

How do you change the CSS for a web page displayed on a phone?

To change the CSS for a web page on a phone, use media queries to apply styles only when the screen size is small.
