What is AJAX?

AJAX is a way for a web page to get or send data from a server without reloading the whole page, making websites faster and smoother to use.

Why do you put your JS Code in a separate file?

You put your JavaScript code in a separate file to keep your code organized, reusable, and easier to manage or update across multiple web pages.
