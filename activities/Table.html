Engagement questions

When do you use a table?

Use a table in HTML to display data in rows and columns for better organization and readability.

When do you not use a table?

Not use a table for layout or styling purposes; instead, use CSS for those tasks.

When do you need to do extra work for table accessibility?

We need extra work for table accessibility when the table is complex or hard to understand, like adding headers.
