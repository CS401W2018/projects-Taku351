Engagement questions

Why don't we use the border attribute?

Because styling should be done using CSS for better flexibility, control, and modern web practices.

What is the easiest way to color every other row a different color?

Use tr:nth-child(even) { background-color: #f2f2f2; } in CSS to color every other row.
