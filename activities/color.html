What is the difference between an ID and a Class?

ID selects only one attribute, but Class will select more than one as a group.

What is the difference between a SPAN tag and a DIV tag?

The SPAN tag is an inline element used for styling small portions of text, while the DIV tag is a block-level element used for structuring and grouping content in a webpage layout.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV is a basic container, while HEADER, MAIN, and FOOTER define the page's header, content, and footer.
