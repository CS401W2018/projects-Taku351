The form tag has an "action" attribute. What does it do?

The action attribute in a form tag specifies the URL where the form's data is sent when submitted.

The form tag has a "method" attribute. What does it do?

The method attribute in a form tag specifies how the form's data should be sent to the server, either using "GET" (appends data to the URL) or "POST" (sends data in the request body).

What is the difference between a "POST" and a "GET"?

GET sends data in the URL, while POST sends data in the request body.
