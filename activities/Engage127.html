Explain the difference between an absolute link and a relative link. Why would you use relative links?

The absolute link connects to the website which is not even in the same file, but the relative link will connect it to the website which is in the same folder.

List the 3 rules with relative links.

We can just put the name of the file into the link and we can go to the web site. And if we put ".." before the file name, we can go up to the file. "../../" and "/" also works.
